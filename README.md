# 🧞‍♂️ 구매탄시장 AI 마법의 족보 (Magic Prompts)

**"복사만 하세요! 당신도 이제 마케팅 천재입니다."**

이 프로젝트는 **수원 구매탄시장 상인분들**이 생성형 AI(ChatGPT, 뤼튼, Gemini 등)를 쉽고 강력하게 활용할 수 있도록 돕는 **웹 기반 프롬프트 라이브러리**입니다.

## ✨ 주요 기능

1.  **원클릭 복사**: 원하는 상황(비 오는 날, 세일, 사과문 등)을 찾아 버튼만 누르면 준비 끝!
2.  **MZ 마법 모드 (Magic Mode)**:
    *   단순한 문구가 아니라, 최신 **프롬프트 엔지니어링**이 적용된 '고성능 프롬프트'로 변신합니다.
    *   **SEO 최적화**, **인스타그램 감성**, **바이럴 유도** 요소를 자동으로 주입합니다.
3.  **모바일 최적화**: 장사하시면서 핸드폰으로 바로 접속해서 쓸 수 있도록 디자인되었습니다.
4.  **힙트로(Hip-tro) 디자인**: 전통시장의 정겨움과 현대적인 힙함을 섞은 차별화된 UI.

## 🚀 사용용

1. 이 페이지에 접속합니다. (링크 예정)
2. 원하는 상황을 검색하거나 카테고리에서 고릅니다.
3. **'MZ 마법 모드'** 스위치를 켭니다. (강력 추천!)
4. **'마법 프롬프트 복사'** 버튼을 누릅니다.
5. ChatGPT나 뤼튼에 붙여넣기만 하면, 마케팅 전문가가 쓴 듯한 글이 나옵니다.
